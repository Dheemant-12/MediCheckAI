# MediCheck AI

An AI-powered medical symptom analysis platform built with FastAPI, React, PostgreSQL, and JWT Authentication.

## Features

* User Authentication
* AI Symptom Analysis
* Multi-Session Conversations
* Conversation Search
* PDF Report Export
* User Analytics Dashboard
* Profile Statistics
* Session Management
* PostgreSQL Database Storage

## Tech Stack

### Backend

* FastAPI
* SQLAlchemy
* PostgreSQL
* JWT Authentication
* ReportLab

### Frontend

* React
* Axios
* Vite
